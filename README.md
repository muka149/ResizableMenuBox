# Resizable Menu Box
![Example](https://github.com/muka149/ResizableMenuBox/blob/main/ArrangementExample_RMB.png)

## 使い方
### フォルダ管理
.uprojectとおなじ場所にPluginsフォルダを展開し、その中にMenuUIフォルダを入れてください

### 基本操作
スプリッターでメニュー間の領域を調整することができるシンプルなメニューUIです

### UEエディタでの使用方法
**スロット階層**

パネル系統のウィジェットと同様に配置できます

![WidgetTree](https://github.com/muka149/ResizableMenuBox/blob/main/WidgetTree_RMB.png)

**パラメータ**

![Param](https://github.com/muka149/ResizableMenuBox/blob/main/Param_RMB.png)

**・Split Type**

分割方向を縦か横に指定します

**・Split Ratio**

スプリット比率を調整します
